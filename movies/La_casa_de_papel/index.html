    《纸钞屋》

    L 7/21/2021 悬疑动作

    推荐

    很多意想不到的点，人物细节丰富，剧情吸引人

    # 评分

    《纸钞屋》目前有四季，第五季在2021年下半年上映。

    豆瓣评分每一季都比上一季要高，第四季高达9.1，期待第五季的收官之战。

    # 一二季：抢劫造币厂

    一二季连在一起主要讲了抢劫造币厂，不动已经印好的钱，自己印钱。那么，这算抢劫么？😂

    从经济学上讲，借用（抢劫）了别人的生产工具，在一段时间内疯狂生产，然后再还回去。从法律上讲，孳息也不是自己的，恶意抢夺还是抢劫。所以这部剧不能模仿哈~

    再回到经济学，第二季“教授”对“督察”说的话也非常有道理。国家一年会印好多好多钱，而这些钱大部分都进入了有钱人的手里（马太效应）。他们只是自己印钱，然后把钱收入自己兜里而已，并没有造成别人的损失。Emmm...确实没毛病。

    但是，国家经济的发展过程肯定要印钱的，要么GDP怎么办，股市怎么办。通胀，众所周知世界各地的通胀是由于美国的货币宽松政策。不印钱你国家外贸怎么办。。。印多了你货币就废了（哦，人家是欧盟。。）。所以“教授”的这种高智商行为，其实是给国家上层人员出了道难题，也就理解了为何第三季开头情报局那么恨他们。

    然而精彩的不只是目标，过程，对人性的把握，套路都异常精彩！艺术方面更棒！即使我们在上帝视角，但还是站在“教授”一方，为他们捏了把汗。

    演员演技也很到位，人性诠释的很棒。有的剧情让人恨得牙根痒痒。

    甚至斯德哥尔摩把人质都策反了。。。“督察”也策反了。。。牛逼！“教授”和“督察”的联手也增加了三四季的看点。

    # 三四季：抢劫国家储备金

    这季的开头确实惊艳了很多，“教授”还是高智商的为组员做生存计划，还是那个味道我就放心了。为了报复情报局对组员的私刑，“教授”决定去抢国家储备黄金。

    看到"教授"说抢黄金的时候我就已经猜到是国家储备金了。。。好家伙

    世界经济，货币只是一张纸。而货币有价值是因为政府的公信力（信用），即人们相信政府不会破产（突然跑路）。而在国际上如何证明自己（国家）货币的价值呢？那就是抗硬通胀的黄金。

    所以"教授"的决定不仅仅是抢劫黄金这么简单，他是在威胁国家破产。。。

    这部剧唯一的槽点就是西方感情戏特别多，出问题基本都是感情和冲动，可能西方文化就是这样的吧。

    # 写在最后

    西班牙题材的影视作品比较出名的特点就是“高智商”，比如《看不见的客人》。这部《纸钞屋》同样延续了风格，西班牙语的语速还很快，拍犯罪片挺带劲的哈哈。

    可能会快进，但不会后悔的一部剧。
