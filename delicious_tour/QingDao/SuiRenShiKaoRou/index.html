    【青岛·燕儿岛路】燧人氏烤肉

    L 7/18/2021 青岛烤肉

    推荐理由

    肉香！服务棒！吃完还不拉肚子！

    美团的猪肉套餐129，牛肉套餐159。最近新店开业更便宜了。。。

    以下，多图预警！！

    # 地址

    2号线燕儿岛路地铁站A口出就是，他是两个门头，两边都可以。

    # 浓浓复古风的装修

    这是两个店内分别拍的

    别有一番风味~

    连卫生间都是一股子古风。。。

    # 逼格高

    之前吃的猪肉套餐特别香，凉面也特别好吃。他家的蘸料是一绝，酸酸甜甜解腻下肚。

    这次是抢到新店开业活动，牛肉套餐。虽然逼哥很高，不过还是推荐去吃猪肉的。烤肉还是猪肉香！

    来一张正面的~ (^▽^)

    # 开烤！

    牛肉如果不是“大师级”的烧烤师傅，就容易烤的很柴。。。😂

    不过我最喜欢吃的还是烤鸡腿肉！🙋

    以及烤猪王排~ 😍

    # 味道超级棒！服务也很棒

    第一次去吃的时候几乎是服务员小哥全程给我们烤的，那口感绝了！搭配店家自己调的秘制酱料，很快我们就吃撑了。。。😂

    最后离开的时候我们把生肉都烤熟了打包带走的哦！店家还送了酱料和包装盒。

    第二次是我自己烤的，后来我想估计是第一次我烤大了，老板看不下去了安排专属服务员帮我烤。。。

    # 最重要的一点！

    青岛这边我吃了很多家烧烤了，他家是掰手指能数出来的吃完了不拉肚子的烤肉店。无论肉是不是冷冻肉，质量最起码放心。

    美团套餐2-3人，两个人吃不上，三个人刚刚好。无论从性价比、口味、地理交通还是食品安全，都是值得去尝一尝的。

    美团评分4.9！
